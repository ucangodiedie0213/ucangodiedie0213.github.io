圖的種類

圖的種類不多 但最常用的就是前兩個

  • 有向圖 : 圖的邊皆為有向邊(每條邊都有方向)
  • 無向圖 : 圖的邊皆為無向邊(每條邊都沒有有方向性)
  • 混合圖 : 同時存在有向邊跟無向邊
  • 簡單圖 : 沒有重邊、沒有自環
  • 完全圖 : 所有點戶相連接
  • 二分圖 : 把一張圖分成兩半，彼此不相連

名詞與專業術語介紹

  • DAG : 有向無環圖

圖的表示法

相鄰矩陣

使用二維矩陣實現，請依照題目選取適合的矩陣型態
如果不帶權重只需要紀錄是否連通，可以用bool，帶權重者可用int實現。
空間複雜度為O(N^2) 但可以做到**O(1)**查詢是否連通

相鄰串列

使用一維vector實現

  • 補充 : 宣告vector時用v[i]是宣告i個vector，但，v(i)則是宣告一個大小為i的vector
    第i個vector存與i點連通的點

圖的遍歷

DFS 深度優先搜尋

DFS根本概念就是有路就走到底，走到死路再回頭去，走另外一條路再走到底。
記得，每次走過點之後就要把它紀錄成有走過的點。
d626: 小畫家真好用
d165: 八、草场普查
DFS是一種遞迴，可以來解排列組合，包牌問題，有些題目會需要剪枝，將不必要的解剪掉。
也因為他是遞迴，寫DFS一定要注意幾個關鍵:邊界、遞迴式、跳出條件
d115: 數字包牌
f168: m4a2-分配寶物(Treasure)
練習題:

最短路徑

最短路有個很關鍵的概念就是鬆弛(Relax)
a->b 的距離如果大於a->c->b的距離就把a->b的距離更新成a->c->b的距離

Bellman-Ford

一種單源最短路徑演算法。
使用相鄰矩陣存圖，無法通過的設置成無現大
在開一個陣列存點對所有點的距離
檢查每條邊是否能鬆弛這個點可以就更新。
如果檢查V(點數)還能夠鬆弛代表有負環。
時間複雜度為O(VE)
洪春男老師講解

SPFA

SPFA(Shortest Path Faster Algorihm)是Bellman-Ford的優化
把更新過的點放入佇列裡只檢查佇列裡中結點上的邊
一樣可以檢查負環
期望複雜度為O(2E)但通常很容易到O(VE)

for(int i=1;i<=n;i++)
    dis[i]=inf;
queue<int>q;
q.push(1);vis[1]=1;dis[1]=0;
while(q.size()){
    x=q.front();q.pop();vis[x]=0;
    for(int i=head[x];i;i=a[i].next){
        int s=a[i].to;
        if(dis[s]>dis[x]+a[i].cost){
            dis[s]=dis[x]+a[i].cost;
            if(vis[s]==0){
            vis[s]=1;
            q.push(s);
            }
        }
    }
}

Dijkstra

洪春男老師講解
用Greddy的方法解決最短路問題，但不能處理有負邊負環的圖。
每次挑選一個離起點最近的點，把它標記成確定最短路徑點後，在利用者個點去鬆弛其他還沒有確認的點。
這裡提供佇列優化版的程式碼，時間複雜度O(nlonn)
程式碼轉自[Zrn-code](https://zrn-code.github.io/2020/09/29/dijkstrat mp/)

typedef pair<int,int>PII;
const int MAXN = 100010;
vector<PII> G[MAXN];
void add_edge(int u,int v,int d){
    G[u].push_back(make_pair(v,d));
}
void init(int n){
    for(int i=0;i<n;i++)
        G[i].clear();
}
int vis[MAXN];
int dis[MAXN]; 
void dijkstra(int s,int n){
    for(int i=0;i<n;i++)vis[i] = 0;
    for(int i=0;i<n;i++)dis[i] = (i == s ? 0 : INF);
    priority_queue<PII,vector<PII>,greater<PII> >q;
	q.push(make_pair(dis[s],s));
    while(!q.empty()){
        PII p = q.top();
        int x = p.second;
        q.pop();
        if(vis[x])continue;
        vis[x] = 1;
        for(int i=0;i<G[x].size();i++){
            int y = G[x][i].first;
            int d = G[x][i].second;
            if(!vis[y]&&dis[x] + d < dis[y]){
                dis[y] = dis[x] + d;
                q.push(make_pair(dis[y],y));
            }
        }
    }
}

樹

可以把樹看成一張沒有環的連通圖

名詞介紹

這裡會介紹一些常用的名詞，但不只這些。

  • 根 : 樹的起點，沒有父節點的點
  • 葉 : 樹的終點，沒有子節點的點
  • 祖先 : 父節點到根的路徑上所有的點
  • 子代 : 向下所有可到的節點
  • 層 : 一個節點與根的距離
  • 深度 : 整棵樹的層數
  • n元樹 : 一個節點最多能擁有n個小孩

二元樹

顧名思義是一個節點最多只能擁有2個小孩的樹。
因為很多資結還有其他應用，所以這裡單獨拿出來討論。

二元樹的遍歷

  • 前序 : 根->左子樹->右子樹
  • 中序 : 左子樹->根->右子樹
  • 後序 : 右子樹->左子樹->根

最小生成樹

生成樹是一顆包含圖上所有點的樹。
最小生成樹則是求圖上權重和最小的生成樹。
有兩種演算法，以邊出發用Kruskal，以點出發用Prim

Kruskal

將邊排序後，由權重最小的邊開始加入生成樹。
使用disjoin set維護是否已經加入樹了(不會用的可以去看基礎資結)
時間複雜度O(ElogE)


模板轉自日月卦長的模板庫

template<typename T>
class kruskal{
	static const int MAXN=100005;
	int n; // 1-base
	tuple<T,int,int> edge;
	int pa[MAXN];
	int find(int x){
		if(x==pa[x]) return x;
		return pa[x] = find(pa[x]);
	}
public:
	void init(int _n){
		n = _n;
	}
	void addEdge(int u,int v,const T &cost){
		edge.emplace_back(cost, u, v);
	}
	pair<T,int> solve(){
		T ans = 0;
		int component = n;
		for(int i=1; i<=n; ++i) pa[i] = i;
		sort(edge.begin(), edge.end());
		for(const auto &e: edge){
			int u = find(get<1>(e)), v = find(get<2>(e));
			if(u==v) continue;
			pa[u] = v;
			ans += get<0>(e);
		}
		return {ans, component};
	}
};

Prim

每次尋找離目前生成樹最近的點加入

模板轉自日月卦長的模板庫

template<typename T>
class prim{
	static const int MAXN=100005;
	struct edge{
		int u, v;
		T cost;
		edge(){}
		edge(int u,int v,const T &cost):u(u),v(v),cost(cost){}
		bool operator< (const edge&b)const{
			return cost > b.cost;
		}
	};
	int n;// 1-base
	vector<edge> E;
	vector<int> G[MAXN];
	bool vis[MAXN];
	T build(int x){
		T ans = 0;
		priority_queue<edge> q;
		for(auto i: G[x]) q.push(E[i]);
		vis[x] = true;
		while(q.size()){
			edge e = q.top(); q.pop();
			if(vis[e.v]) continue;
			ans += e.cost;
			vis[e.v] = true;
			for(auto i:G[e.v]) if(!vis[E[i].v]){
				q.push(E[i]);
			}
		}
		return ans;
	}
public:
	void init(int _n){
		n = _n;
		for(int i=1; i<=n; ++i) G[i].clear();
	}
	void addEdge(int u, int v, const T &cost){
		G[u].emplace_back(E.size());
		E.emplace_back(u, v, cost);
		G[v].emplace_back(E.size());
		E.emplace_back(v, u, cost);
	}
	pair<T,int> solve(){
		T ans = 0;
		int component = 0;
		for(int i=1; i<=n; ++i) vis[i] = false;
		for(int i=1; i<=n; ++i) if(!vis[i]){
			ans += build(i);
			++component;
		}
		return {ans, component};
	}
};

拓樸排序

給一DAG，使所有序號大的點不可到序號小的點;所有序號小的點不可到序號大的點。
通常搭配stack存取，這並不一定。
每次將出度為0的點push進stack裡即可
再將這個點的所有小孩出度-1
