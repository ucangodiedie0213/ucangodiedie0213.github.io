先備知識

安裝軟體

anaconda

下載並安裝，如果他沒有裝python需要自己裝
Anaconda官網

qiskit

開啟anaconda prompt
輸入以下指令

pip install qiskit

安裝完後輸入以下指令開啟jupyter

jupyter notebook

它會在瀏覽器打開jupyter
點選new然後選python3

量子初體驗

首先我們可以先查看Qiskit版本

import qiskit
qiskit.__qiskit_version__

就可以查看所有目前安裝的qiskit版本
可以和Github上的版本對照

取得IBMQ的API Token

因為我們現在是用經典電腦模擬量子電腦，速度和機率、情況等等都跟真實的電腦不同
等等會示範一個範例，來看經典與量子的差別。
但是我們沒有真正的量子電腦，所以我們必須仰賴IBM的機器，將我們的工作加入隊列，在回傳答案給我們。
我們需要有IBM的Token我們才有辦法把我們的工作丟給IBM。
我們需要先在IBM Quantum Experience註冊一個帳號，取得API Token
IBM Quantum Experience
登入後，你可以在backend看到目前可以使用的量子電腦的狀況。
然後在右上角My account中可以看到自己的Token，把它複製下來
在jupyter上輸入以下指令。引入IBMQ，儲存token，讀取帳號。

from qiskit import IBMQ
IBMQ.save_account('[token]')
IBMQ.load_account()

此時他會顯示你帳號信息，有可能會報錯因為時區不同
這樣你就可以開始玩量子了~
