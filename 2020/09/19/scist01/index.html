Linux

Linux 是一個類unix 免費且開源的操作系統
在你CTF的路上他可是你不可或缺的好夥伴呦
趕快跟Linux交個朋友吧!

Commands

  • ls
    執行ls指令會列出當下目錄的所有檔案
    也可以ls指定目錄 只要在ls後加上路徑
    ls [DIR]
    可選參數若填入-al 可列出所有檔案的詳細資訊

ls -al
星號為萬字元
使用者列表位於/etc/passwd
密碼存於/etc/shadow

  • chmod
    chmod 可以更改檔案權限
    權限分為擁有者、群組、所有人
    這三個都各有rwx三個參數
    可用二進制表示
    rwx為7(111) r-x為5(101) rw-為6(110)

chmod 777 file

更改資料夾 -R

  • sudo
    以root權限執行
    當你執行大部分的系統命令時出現沒有權限，可在指令前加入sudo解決

sudo su 可直接將自己變成root

  • wget

wget [網址] [參數]
參數常用為-O [檔案名稱] 可將下載下來的檔案直接命名

  • file

file [FILE]
取的檔案型態
ELF為可執行檔
ASCII txt為文字檔可直接cat
gzip壓縮 可用tar解壓

  • cat
    將檔案內的內容吐出來

cat [FILE]

輸出型態

輸出分為 STDIN(0) STDOUT (1) STDERR(2)
有時候會有用STDERR偽裝STDOUT
需要過濾

  • apt
    安裝程式

  • tree
    美化版ls

    -L 可指定最深幾層

tree -L 2

  • cd
    移動目錄

    /為根目錄
    ~預設為home
    .為當前目錄
    …為上一層

  • tar
    壓縮打包與解壓
    參數
    z為用gzip操作;x為解壓縮;c為壓縮;v詳細資料;f指定檔名
    解壓範例

tar zxvf [FILE]

  • grep

grep KEYWORD [FILE]
搜尋關鍵字

管道符號與重導向符號

command1 | command2
把command1的STDOUT當成command2的STDIN執行
command1 > file
將command1的STDOUT寫入file
command1 >>file
將command1的STDOUT寫入到file的後方

  • touch

touch [檔名]
建立新檔案

  • mkdir

mkdir [資料夾名稱]
建立新資料夾

  • mv
    移動檔案

mv [FILE] [DIR]

  • cp
    複製檔案到路徑

cp [FILE] [DIR]

  • rm
    刪除

rm [FILE]

  • strings
    吐出檔案內所有可視字元

strings [FILE]

  • md5sum
    計算md5值

md5sum [FILE]

  • base64
    使用base64做編碼解碼
    預設為編碼 解碼只要加上-d的參數

base64 -d [FILE]

  • ssh
    遠端連線電腦

ssh user@ip

若要指定port
加上-p [PORT] 參數
  • find
    在指定目錄尋找檔案名稱

find [PATH] -name [FILENAME]

  • service
    對特定服務進行操作

service [NAME] [start|stop|status|restart]

  • ps -aux
    查看所有服務
  • curl
    模擬人為操作瀏覽網站

curl [host/ip]
