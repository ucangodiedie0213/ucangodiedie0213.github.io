a129.最小生成樹

2020年05月05日 485 字 大概 2 分钟
  1. 1. 內容
  2. 2. 輸入說明
  3. 3. 輸出說明
  4. 4. 解題思路
  5. 5. 程式碼

內容

給你一個加權的無向圖(weighted undirected graph)，請問這個圖的最小生成樹(minimum spanning tree)的權重和為多少？

輸入說明

輸入可能包含多筆測試資料，以EOF作為結束。

每筆測試資料的第一列有兩個整數n,m(1<=n<=100,000；0<=m<=200,000)，代表該圖的點數和邊數。
頂點的編號從0到n-1。
接下來有m列，每列用三個整數i,j,c(0<=i,j<n c為int可儲存的非負整數)描述一條邊，i,j為兩個端點的編號，c為其權重。

輸出說明

對於每筆測試資料，請輸出最小生成樹的權重和。如果圖不連通，請輸出-1。

解題思路

本題的想法是使用最小生成樹和並查集完成的
father是用來記錄父節點的;num是在記錄樹的大小以便合併
將存有各邊數據的vcetor排序依照權重由小到大
for跑 找根 如果根不同 就將權重加入 並合併
還要順便確認邊的數量是否相同

程式碼

#include<bits/stdc++.h>
using namespace std;
int father[100000],num[100000];
struct node {
int start,stop,length;
};
int find_father(int s){
    while(father[s]!=s){
        s=father[s];
    }
    return s;
}
bool myCompare(node a,node b){
return a.length<b.length;
}
int main()
{
    int n,m;
    while(cin>>n>>m){
        vector<node> v(m);
        long long int ans=0;
        for(int i=0;i<n;i++){
            father[i]=i;
            num[i]=1;
        }
        for(int i=0;i<m;i++){
            cin>>v[i].start>>v[i].stop>>v[i].length;
        }
        sort(v.begin(),v.end(),myCompare);
        int edgesum=0;
        for(int i=0;i<m;i++){
            int a,b;
            a=find_father(v[i].start);
            b=find_father(v[i].stop);
            if(a!=b){
                if(num[a]>num[b]){
                    father[b]=a;
                    num[a]+=num[b];
                    ans+=v[i].length;
                    edgesum++;
                }
                else{
                    father[a]=b;
                    ans+=v[i].length;
                    num[b]+=num[a];
                    edgesum++;
                }
            }
        }
        if(edgesum==n-1)cout<<ans<<endl;
        else cout<<-1<<endl;

    }
    return 0;
}
