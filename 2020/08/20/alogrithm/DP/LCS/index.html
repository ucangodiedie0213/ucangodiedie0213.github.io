最長共同子序列-LongestCommonSubsequence

  • DP

2020-08-20

LCS

s1 = 2 5 7 9 3 1 2
s2 = 3 5 3 2 8
LCS(s1,s2) = 5 3 2

LCS可能有很多個

s1: a b c d b c e e a
s2: c a b d e f g a
s3: d c e a
LCS(s1, s2, s3) = {c e a, d e a}

想法

DP的精隨就是要找到子問題
所以我們要把題目變簡單
把序列拆成最後一項跟剩下的

s1 = sub1+e1
s2 = sub2+e2

這時候的LCS就有四種情況

  • 包含e1 不含 e2
  • 不含e1 包含e2
  • 不含e1 不含e2
  • 包含e1 包含e2

再來我們來討論LCS會在哪裡

1.e2沒用 所以LCS會在sub2裡面 故 LCS(s1,s2)=LCS(s1,sub2)
2.同上 LCS(s1,s2)=LCS(sub1,s2)
3.兩者皆沒用 所以LCS在sub1,sub2裡面 LCS(s1,s2)=LCS(sub1,sub2)
4.同時包含故LCS最後一個元素必定是e1(e1==e2) 故 LCS(s1,s2)=LCS(sub1,sub2)+e1

整理公式並簡化

LCS(s1,s2)=
{ if(e1!=e2) max( LCS(sub1,s2),LCS(s1,sub2) )}
{ if(e1==e2) LCS(sub1,sub2)+e1}
