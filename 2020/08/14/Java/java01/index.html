變數與常數

整數

大部分使用int就可以偶爾會用到byte

型態 空間(位元組) 範圍
byte 1 -128~127
sort 2 -32768~32767
int 4 -2147483648~2147483647
long 8 -9,223,372,036,854,775,808~9,223,372,036,854,775,807

浮點數

浮點數可以儲存與小數兩個視範圍與用途選擇
在程競的世界，在需要用浮點數做運算的時候會盡量使用double來避免浮點數誤差
所以，只要有需要運算盡量使用double

型態 空間(位元組) 範圍
float 4 3.410^-38~3.410^38
double 8 1.710^-308~1.810^308

布林值

布林值可以存取true或者false(數字分別代表1和0)
型態:boolean
空間(位元組):4
範圍:0~1

字元

字元能用來儲存一個字預設值null
型態:char
空間(位元組):2
範圍:ASCII碼

字串

字串能用來儲存多個字
型態:String

常數

在Java的世界可以用final 型態來宣告一個不可以改變的常數
